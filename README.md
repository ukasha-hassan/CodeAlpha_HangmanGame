# Hangman Game 🎮

A simple Hangman game implemented in Python.

## 📜 Description

This is a console-based version of the classic Hangman game. The computer randomly selects a word from a list, and the player has 6 chances to guess the word by suggesting letters.

## 🛠️ Features

- Random word selection
- Limited attempts (6 turns)
- Real-time feedback on guesses
- Console-based gameplay

## 📂 How to Run

1. Install Python (if not already installed): [python.org](https://www.python.org/)
2. Clone this repository or download the file.
3. Open terminal or command prompt.
4. Run the game with:

```bash
python hangman.py
